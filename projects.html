Stock Editor

Stock Editor is a desktop inventory management application built with Qt/C++ that provides businesses with a streamlined solution for product tracking and stock control. The application features an intuitive interface for managing product details including barcodes, descriptions, stock levels, and pricing. Users can easily import and export inventory data, modify existing items, and track stock levels through a clean tabulated display. Built with modern C++ and the Qt framework, Stock Editor demonstrates robust software engineering principles while maintaining a user-friendly experience through its toolbar-based navigation and responsive layout. The application combines efficient data management with a professional interface, making inventory control accessible and reliable for business users.

Gauteng locations distance calculator

This C++ program implements a graph-based system to find the shortest path between locations in Gauteng, South Africa, using Dijkstra's algorithm. The program defines a graph with 103 nodes, each representing a location, and edges representing the distances between them. The Node structure is used to create a linked list for each node, representing its connections to other nodes. The create_gauteng_graph function initializes the graph with predefined connections and distances. Users input a start and end location, and the program calculates the shortest path using the dijkstra function, which returns the shortest distances and the previous node for each node in the path. The reconstruct_path function is used to trace back the shortest path from the end node to the start node, and the print_shortest_path function displays the path and its total distance. The program also maps node indices to location names for user-friendly output. This program could be used for applications such as route planning, navigation systems, or educational tools to demonstrate graph algorithms.
